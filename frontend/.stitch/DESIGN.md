# Booking Family Admin Design System

**DESIGN SYSTEM (REQUIRED):**
- **Platform:** Web, Desktop-first
- **Palette:** 
  - Primary: #468faf (CTA, active states, highlights)
  - Secondary: #89c2d9 (Soft accents)
  - Dark: #013a63 (Sidebar background, dark text)
  - Background: #f8fbfd (Main app background)
  - Card/Surface: #ffffff (White cards/panels)
- **Typography:** 
  - Font Family: Roboto mono
  - Heading: 20-24px
  - Body: 14-16px
  - Table: 14px
- **Styles:** 
  - Border radius: 12px for all cards, modals, and buttons.
  - Grid spacing: 16-24px between elements.
  - Cards: Equal height, clean white background `#ffffff`, subtle shadow/lift effect on hover.
  - Tables: Clean layout, minimal heavy borders, well-spaced rows.
- **Interactions & States:**
  - Sidebar Active Item: Background `#468faf`, Text `white`.
  - Hover on Cards: Add subtle shadow and slight lift.
  - Buttons: Scale down slightly when clicked.
  - Modals: Fade and slide up animation.
  - Status Badges: Pending (Grey), Doing (Green `#89c2d9` or similar), Done (Darker Green or Primary `#468faf`).
  - Forms: Realtime validation context, loading spinners, disable button on submit, success toast on completion.
